# nl-childcare-lrk-kvk-ghost-providers

Cross-reference LRK childcare registry with KVK company data and BAG address
registry to identify potential ghost providers.

## Requirements

```
pip install -r requirements.txt
```

## Run

```bash
python3 pipelines/nl-childcare-lrk-kvk-ghost-providers/analyze.py
```

Outputs are written to `pipelines/nl-childcare-lrk-kvk-ghost-providers/output/`.
