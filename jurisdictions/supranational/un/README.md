# United Nations (UN)

## Context

UN system procurement and aid transparency portals:

- **UNGM** (United Nations Global Marketplace) — Procurement portal for the UN system.
- **IATI** (International Aid Transparency Initiative) — Standardized data on development aid flows.
- **UN Procurement Division** — Annual Statistical Report on UN procurement.

## Relevant frameworks

- UN Procurement Manual (Rev. 7)
- IATI Standard (v2.03)
- UN Financial Regulations and Rules

## Focus areas

- Vendor concentration in UN agency procurement
- Aid flow routing anomalies
- Procurement from sanctioned entities or jurisdictions
